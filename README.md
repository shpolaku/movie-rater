## Usage

1. Enter a movie name
2. Click the stars to rate it (1-5 stars)
3. Click "Add Movie" to save
4. View all your rated movies below
5. Delete any movie by clicking the delete button

## Technologies

- No dependencies or frameworks needed
- LocalStorage API for data persistence
